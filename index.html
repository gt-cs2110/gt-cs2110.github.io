System Requirements

Supported Distros: Ubuntu 16.10 (yakkety), 16.04 (xenial), or 14.04 (trusty) OR any distro depending on the aforementioned Ubuntu versions is fine (Linux Mint, *buntu, etc).  Either running through VMWare or Natively.

At least 1GB of ram more is better

Installing Complx

These commands will only work on the above mentioned supported distros.

It is HIGHLY RECOMMENDED you follow these instructions and only these instructions, because I will be releasing an update to complx within a week.

Open a Terminal and run the following 3 commands.

  • sudo add-apt-repository ppa:tricksterguy87/ppa-gt-cs2110
  • sudo apt-get update
  • sudo apt-get install complx-tools

Please run complx at least once (simply execute complx within a terminal window), examine the UI and report any issues to your TA.  The main window should look like this.

Be sure to read the documentation available here

Installing Complx via its source code

If you don’t satisfy the above requirements for supported distros above please follow the instructions here for compiling complx from source manually.

These instructions should work for bash on ubuntu, and probably macs.  However use of these systems are highly discouraged due to being untested on these platforms, people have reported crashes on these systems so YMMV.  I will not be entertaining any bug reports from these systems.
